ğŸ› TROUBLESHOOTING:
-------------------
1. Port in use:
   ```bash
   sudo lsof -i :5000
   # Change port in app.py if needed
   ```

2. LXD issues:
   ```bash
   sudo systemctl restart snap.lxd.daemon
   sudo lxd init
   ```

3. Permissions:
   ```bash
   sudo usermod -aG lxd $USER
   newgrp lxd
   ```

4. tmate not found:
   ```bash
   lxc exec <name> -- apt update
   lxc exec <name> -- apt install tmate -y
   ```
